# Desktop end-to-end coverage

The production image is exercised at desktop viewport sizes. The critical flows are:

- open a source and start a metadata scan;
- select the Rename Only preset and toggle a processor;
- resolve review items and inspect their decision trace;
- freeze a virtual plan, confirm execution, pause safely, and simulate rollback;
- switch between English and Simplified Chinese resources.

No mobile-specific workflow is supported.
